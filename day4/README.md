# Sneaky Extra Skill
<br />

## syntax 
**print** ("\033[colour value m", "**TEXT**")

## Example
print ("\033[31m", "Warning", "\033[32m", "Safe")<br />
here "Warning" will be printed in red colour and "Safe" will be printed in green colour.


| Colour    | Value     |
|-----------|-----------|
| Default   | 0         |
| Black     | 30        |
| Red       | 31        |
| Green     | 32        |
| Yellow    | 33        |
| Blue      | 34        |
| Purple    | 35        |
| Cyan      | 36        |
| White     | 37        |
